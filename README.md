# SKYBY - WiFi Scan Bypass Tool

## Features
- **Option 1 (WiFi Setup):** Auto-detect WiFi Name & Router IP.
- **ADB Persistence:** WiFi မပြောင်းသရွေ့ ADB reconnect လုပ်စရာမလို။
- **Auto Reconnect:** လိုင်းကျရင် အလိုအလျောက် ပြန်ချိတ်ပေးမယ့်စနစ်။
- **Option 5 (Get Portal Link):** WiFi Login Link ကို အလိုအလျောက် ရှာဖွေပေးခြင်း။
