# SKYBY - WiFi Scan Bypass Tool

## Installation
```bash
cd $HOME && rm -rf skyby && git clone https://github.com/paingzin3521-ux/start skyby && cd skyby && bash install.sh && skyby
```

## Features
- Auto WiFi SSID & Router IP detection
- ADB Persistence (No need to reconnect on same WiFi)
- Auto Reconnect & Keep-Alive
- Option 5: Get Portal Link
